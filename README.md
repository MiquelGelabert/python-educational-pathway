# Python Educational Pathway: From Basics to Logic

Note: The notebook and exercises in this repository are written in Spanish.

This repository is a curated collection of exercises and examples designed to teach Python programming. It follows a pedagogical progression, starting with basic syntax and gradually increasing in difficulty to build a solid foundation in coding logic.

## About this Project

I created this resource because I am passionate about sharing my knowledge and teaching others. I believe that the best way to truly master a concept is by explaining it simply to someone else. This repository serves as my personal teaching guide; it is designed to help beginners navigate the initial hurdles of programming with clear, commented examples and a structured path.

## Learning Roadmap

The content is structured to be explained in a progressive way:

### 1. Foundations
* Basic syntax and data types.
* Control flow: if, while, and for loops.
* Function definition and parameter handling.

### 2. Intermediate Logic
* List and dictionary manipulation.
* Practical exercises involving mathematical sequences and natural numbers.
* Introduction to logical thinking and problem decomposition.

### 3. Introduction to Data Tools
* Basic usage of the Pandas library.
* Managing DataFrames: creating structures, accessing rows/columns, and identifying data types.

## Why this Repository?
* Guided Mentor Mentality: Designed for live explanation or structured self-study.
* Progressive Difficulty: Each section builds upon the previous one to avoid overwhelming the learner.
* Practical Focus: Every concept is tied to a small exercise that demonstrates its immediate utility.
